# Dataset

## NEU Surface Defect Database

**Download:** [Kaggle - NEU Surface Defect Database](https://www.kaggle.com/datasets/rdsunday/neu-urface-defect-database)

**Source:** [faculty.neu.edu.cn](http://faculty.neu.edu.cn/songkechen/zh_CN/zhym/263269/list/index.htm)

**Authors:** Kechen Song and Yunhui Yan (Northeastern University)

**Paper:** *NEU surface defect database*

### Description

1800 grayscale images (200×200 px) of hot-rolled steel strip surface defects — 300 samples per class.

### Setup

After downloading, place the dataset contents in this folder (`data/`).

### Classes

| Code | Defect |
|------|--------|
| RS | Rolled-in scale |
| Pa | Patches |
| Cr | Crazing |
| PS | Pitted surface |
| In | Inclusion |
| Sc | Scratches |
